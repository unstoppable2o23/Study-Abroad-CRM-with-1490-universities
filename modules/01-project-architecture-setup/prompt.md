# Module 1: Project Architecture Setup

## Objective
Set up the foundational project structure for a multi-tenant Study Abroad CRM platform using Next.js 15+ with App Router, Material UI, Prisma ORM, and PostgreSQL.

## Key Requirements
- Next.js 16 with App Router
- Material UI with swappable architecture
- Prisma ORM with PostgreSQL
- Modular directory structure
- Environment configuration
- JWT authentication foundation
- RBAC readiness
- Docker development setup
